# Project Management

Issue tracking best practices for Linear and Jira.

## Skills

| Skill | When to use |
|-------|-------------|
| **linear-method** | Creating/organizing/prioritizing issues in Linear |
| **atlassian-cli-jira** | Installing and using Atlassian CLI for Jira Cloud |

## Install

```bash
claude plugin marketplace add danmestas/agent-skills
claude plugin install project-management
```
