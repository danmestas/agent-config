# Dev Tools

Development and testing utilities.

## Skills

| Skill | When to use |
|-------|-------------|
| **doppler** | Migrating .env files to Doppler secrets management |
| **midscene-testing** | Ad-hoc browser testing, smoke testing UI workflows |
| **deterministic-simulation-testing** | Testing distributed systems with deterministic simulation |

## Install

```bash
claude plugin add danmestas/agent-skills --plugin dev-tools
```
